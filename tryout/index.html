privacymelding

Omdat het niet mogelijk is om direct met Magister te verbinden vanuit de browser hebben wij (simplyApps) een gratis proxy opgezet waarmee je magister.js kan testen. Hierdoor wordt wel alle data verstuurd naar deze proxy, dit gebeurd via een beveiligde TLS verbinding. De proxy draait op Heroku. De code van de proxy is hier beschikbaar, wij draaien deze code ongewijzigd. Er wordt door ons géén data opgeslagen. Als je liever wilt dat je data alleen naar Magister verstuurd wordt kan je het gemakkelijkste magister.js testen met node.js.
