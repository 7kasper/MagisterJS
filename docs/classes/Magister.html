Magister

(
  • options
)

Defined in Magister.coffee:25

Parameters:

  • options Object

    An object containing your perfered options.

    • school MagisterSchool | String

      A MagisterSchool to logon to. If this is a String it will use that String as a query to search for a possible school.

    • [username] String optional

      The username of the user to login to. (Not needed when using an sessionId.)

    • [password] String optional

      The password of the user to login to. (Not needed when using an sessionId.)

    • [sessionId] String optional

      An sessionId to use instead of logigging in to retreive a new one.

    • [keepLoggedIn=true] Boolean optional

      Whether or not to keep the user logged in.
