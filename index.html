Magister.js

A
browserside
implementation of the Magister 6 API

Magister.js is an opensource asynchronous JavaScript implementation of the non-public API of Magister, built by Lieuwe @ simplyApps.
The goal is to create an easy to use and open way to communicate with Magister.

Help by making pull requests and issues at the Magister.js repo.
Code is documented with YUIDoc.

Get started
Node.js
npm install magister.js
Meteor
meteor add simply:magisterjs
Browserside
Download minified JS with README


Examples

Log the full name of the teacher for the first lesson of today.
new Magister({ url: "https://adelbert.magister.net" }, "xxxx", "xxxxx").ready(function (m) {
	m.appointments( new Date(), function (error, result) {
		console.log(result[0].teachers[0].fullName());
	});
});
Send a message to my 'friends', I.. I... don't have friends..
new Magister({ url: "https://adelbert.magister.net" }, "xxxx", "xxxxx").ready(function (m) {
	m.composeAndSendMessage("subject", "body", ["Thomas", "Tom", "Jesse"]);
});
Download the first File in the first FileFolder from Magister.
new Magister({ url: "https://adelbert.magister.net" }, "xxxx", "xxxxx").ready(function (m) {
	m.fileFolders(function (error, result) {
		result[0].files(function (error, result) {
			result[0].download();
		});
	});
});
